Germination

Prior to beginning a germination trail, seeds must be accessioned and properly recorded in the Research Database.

A Biological Asset Transfer Form must also be completed to reflect the removal of seed from storage.

The Research Database Website has a Germination Trial Web Tool to aid in the management of data. THe tool allows the user to select the seeds to be used, methods used, and create new records for any plants produced from the trials.

Consult with the Database Associate if you would like to set up a Germination Trial or need any assistance doing so.
