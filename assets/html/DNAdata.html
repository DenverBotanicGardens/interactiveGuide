Submitting DNA Sample Data

Submitting data for DNA sample records requires 4 steps.

1. Gathering and listing all materials from which DNA will be extracted from. Individual ID in the case of a tissue sample or Catalog Number in the Case of a Specimen is the most relevant information.
2. Check that all tissue samples or specimens are accessioned and have records in the Research Database.
3. Using the online DNA Extraction Project Tool, create a new project and add all materials from which DNA will be extracted from to the project.
4. Complete the web form, being sure to include at minimum the following:
    a. DNA bank number/vial label
    b. Date of extraction
    c. Staff member conducting extraction
    d. If the source tissue is all gone
    e. The position of the vial in its tray


*Steps 1-3 can be completed by Database Associate if a spreadsheet of Individual IDs (tissue samples) or Catalog Numbers (specimens) is provided. The Database Associates only asks that you submit a complete list prior to beginning the extraction process.

**Step 4 can be completed by Database Associate if a spreadsheet containing the listed information is provided, and steps 1-3 have already been completed.
